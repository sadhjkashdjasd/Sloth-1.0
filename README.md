# Sloth-1.0
My own programming lang named sloth. Kind of slow, but shorter then python.

How to setup?
Download
PLace somewhere (Recomandated is "C:/Sloth/")
Add this directory to PATH.

How to run?
Open directory in cmd.
Input `sloth`.
Done.
